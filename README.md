### Guide de démarrage

Côté Backend (Le cerveau)

    Ouvrir un terminal dans /backend.

    Activer l'environnement : source .venv/bin/activate (sur Linux/Mac).

    Installer les outils : pip install -r requirements.txt.

    Lancer : python3 app.py.

Côté Frontend (Le visage)

    Ouvrir un deuxième terminal dans /frontend.

    Installer les outils : npm install.

    Lancer : npm run dev.

    Cliquer sur le lien (souvent http://localhost:5173) qui s'affiche.
